# YouTube to MP3 Converter

A portable Windows desktop utility that downloads YouTube audio and converts it into MP3 files.

## Features
- Graphical user interface using Tkinter.
- Multi-threaded downloads to prevent the application from freezing during execution.
- File dialog to select a custom destination folder for downloads.
- Portable executable that runs without requiring a Python installation on the host machine.

## How to Run
1. Go to the Releases section on the right side of this page.
2. Download the Conv_Windows.zip package and extract it.
3. Open the extracted folder and run Conv.exe (you can create a desktop shortcut for easier access).

## Requirements used for building
- Python
- Tkinter
- yt-dlp
- FFmpeg
